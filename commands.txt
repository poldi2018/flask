To list processses running flask:
ps -fA | grep python

request.form object containing form data, with key:
print(request.form["name"])

upgrade python:
brew upgrade python
(or via installer)

upgrade pip:
pip3 install --upgrade pip

install flask:
pip3 install Flask

running flask app:
python3 -m flask run

set python version in runtime.txt:
echo "python-3.7.4" > runtime.txt

create requirements.txt:
pip3 freeze --local > requirements.txt


HEROKU:

Install the Heroku CLI
Download and install the Heroku CLI.

If you haven't already, log in to your Heroku account and follow the prompts to create a new SSH public key.

$ heroku login
Create a new Git repository
Initialize a git repository in a new or existing directory

$ cd my-project/
$ git init
$ heroku git:remote -a poldis-thorin-and-company-app
Deploy your application
Commit your code to the repository and deploy it to Heroku using Git.

$ git add .
$ git commit -am "make it better"
$ git push heroku master
Existing Git repository
For existing repositories, simply add the heroku remote

$ heroku git:remote -a poldis-thorin-and-company-app

clear buildpacks:
heroku buildpacks:clear

show list of apps:
heroku apps

renaming app:
heroku apps:rename [old_app_name] --app [new_app_name]

checking for set remote locations:
git remote -v

setting remote location:
git remote add heroku https://git.heroku.com/poldis-thorin-and-company-app.git

push to heroku:
git push -u heroku master

generating Procfile:
echo web: python app.py > Procfile

start web service:
heroku ps:scale web=1

config vars in heroku:
IP: 0.0.0.0
PORT: 5000
